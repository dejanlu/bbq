# BBQ 

BBQ restaurant web app.

## Installation

Use the node package manager (npm) to install foobar.

```bash
npm install
```

## Usage

```bash
npm start
npm test
```

## Dependecies

* react
* redux
* firebase
* node-sass
* framer-motion
* react-icons
* react-intersection-observer
